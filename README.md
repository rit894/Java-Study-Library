

Java Study Library
==================

This repository contains a collection of Java programs, utilities, and examples designed to help students learn and practice Java concepts. It is organized by topic and kept simple for easy reference.

Features
--------
- Core Java examples such as object-oriented programming, collections, exceptions, and input/output.
- Utility classes for common tasks like string handling, math helpers, and file operations.
- Demo programs that can be run directly.
- Code is commented and explained for clarity.
- Topics are separated into folders for quick navigation. 

Installation and Setup
----------------------
1. Clone the repository using:  
   git clone https://github.com/your-username/java-study-library.git  

2. Open the project in your preferred IDE such as Eclipse, IntelliJ, or VS Code.  

3. Build and run examples from the src/ folder.  

Usage
-----
Example of running a simple program:

```java
import basics.HelloWorld;

public class Main {
    public static void main(String[] args) {
        HelloWorld.run();
    }
}
```

Goals
-----
- Help students revise Java concepts quickly.  
- Provide reusable snippets for assignments and projects.  
- Encourage learning by example and experimentation.
